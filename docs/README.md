# Valier UI Phase 3B - High Priority Production Assets

This pack refines the first high-value production asset groups:

- Login UI
- Inventory / containers
- Spellbook
- Chat UI
- Action bars / hotbars

## Important
This pack preserves the original source silhouettes and outlines.
It is intended for client integration and hand-tuned polish, not destructive auto-cleanup.

## Included
- named production assets for each priority screen
- state-split button strips
- 9-slice folders for login, chat, and tall container panels
- screen mockups to guide integration

## Recommended integration order
1. LoginUI
2. Chat
3. Inventory
4. Spellbook
5. Hotbars
