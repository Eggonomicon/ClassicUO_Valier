# Valier UI Panel Standards v2

## Goal
Preserve the original silhouette, bevel, and outline of the art. Do not alpha-strip or auto-mask these panels during foundation packaging.

## Panel rules
1. Corners never stretch.
2. Border bands stretch minimally.
3. The center absorbs most scaling.
4. Text and controls stay inside the inner surface, away from gold trim.
5. Safe content padding starts at 16 px minimum and increases by panel size.

## 9-slice defaults
- Main panel: 54 / 50 / 54 / 50
- Side tall panel: 24 / 40 / 24 / 40
- Chat panel: 28 / 32 / 28 / 32
