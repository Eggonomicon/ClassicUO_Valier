# Valier UI Naming Rules v2

## Format
`<category>/<asset_family>_<variant>_<state>.png`

Use concise, stable names. Keep nouns before state names.

## Categories
- Buttons
- Checkbox
- Radio
- Panels
- Frames
- Scrollbars
- Chat
- Hotbars
- Inventory
- Spellbook
- Containers

## State suffixes
- idle
- hover
- pressed
- disabled
- active
- selected
- off
- on
