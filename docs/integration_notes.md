# Valier UI Phase 3A Foundation Pack v2

This v2 pack corrects the Phase 3 preview problem by preserving the original cropped artwork and outlines.

## What changed from v1
- Removed the aggressive auto-cleaning pass that caused colored halos and lost silhouettes.
- Rebuilt the asset folders from the original atlas/sheet.
- Preserved original outlines and edge shading.
- Kept the foundation docs focused on naming, panel standards, and control families.

## Recommended next step
Use this pack as the system foundation, then move to the ValierUO client chat for actual ClassicUO integration and asset wiring.
